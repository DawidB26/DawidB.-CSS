1. Jakie standardy są dostępne w systemie Windows(trzy, obliczenia wyliczeń z kwadratem)

      2. W systemie Windows standardowo można używać czcionek:
    • Times New Roman (szeryfowa),
    • Arial (bezszeryfowa),
    • Kurier Nowy (monotypiczna).

  2. Jakich czcionek dodatkowo dostarcza Internet Explorerem ( cztery, obliczenia wyliczenia z okręgiem nie kołem).

    3. Dodatkowo z Internet Explorerem dostarczane są:
    • Verdana,
    • Tahoma,
    • 'Trebuchet MS',
    • Gruzja.

  3. Wymień rodziny ogólne ( pięć, obliczenia wyliczenia z kołem, podkreślen33ie, przykłady) .

    • serif - czcionka szeryfowa (końcówki znaków danego "ozdobniki"), np.: 'Times New Roman”, Georgia, Garamond, Bodoni
    • sans-serif - czcionka bezszeryfowa (końcówki znaków są proste), np.: Arial, Verdana, „Trebuchet MS”, Helvetica, Univers, Futura
    • monospace - czcionka o użyteczności znaków - monotypiczna (wygląda, jak Pisana na maszynie), np.: Kurier, 'Kurier Nowy'
    • cursive - czcionka określona cechami pochyłej (wygląda, jak pisany ręcznie)
    • fantasy - czcionka fantazyjna (dekoracyjna)

  4. Kiedy stosujemy rodzinę czcionek, w jaki sposób wpisujemy( deklarujemy rodziny czcionek).

    5. Rodziny ogólne wykorzystuje się jako kolejną - alternatywę dla nieużytkownika będzie posiadał dowolną czcionkę, którą stworzysz wprost. Dlatego zwykle rodzina Podaje się na samym końcu listy nazw czcionek

  5. Rodzaje wyrównanie tekstu (cztery, wyliczenie)

    6. Wyrównanie tekstu może być:
    • Zrób rano
    • Do lewej
    • Wyśrodkowane
    • Wyjustowane

  6. Kolor tekstu (składnia opisu)

    7. Kolor tekstu składnia: selektor { kolor : kolor ; } Wyraz selektor ma zastosowanie do wszystkich elementów. Jako kolor należy określić rodzaj koloru. Dzięki temu poleceniu można zastosować zmianę koloru składu.

  7. dekoracja tekstu (składnia, opis, skutki wcięcia, podkreślenia numerowania a) b)…)

    8. Dekoracja tekstu
    Tekst można „udekorować” w jaki sposób:

    a) Podkreślić - (podkreślenie)
    b) Przekreślić - (line-through)
    c) Nadkreślić - (overline)
    d) Można dodać migotanie tekstu (tylko Netspace/Mozilla/Opera) – (blink)

    Składnia:
    selektor { tekst -dekoracja : dekoracja ;}
